# Kohlrabi


This Mod adds Kohlrabi to Minetest.<br>
It supports farming and farming_redo from TenPlus.

![Screenshot 1](textures/kohlrabi_screenshot.jpg)

## Install

Move your Download to the Mods-Folder.

## Depends

default<br>
farming<br>

### optional
farming_redo<br>
bucket<br>
lettuce<br>
parsley<br>
vegan_sunflower<br>
